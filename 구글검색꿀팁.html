<구굴 검색 꿀팁>

1. 특정 단어/문장을 반드시 포함 시킬때

검색어의 처음과 끝에 큰 따옴표("")를 입력하면 해당단어나 문장이 반드시 포함된 검색 결과를 얻을 수 있습니다.


2. 특정 단어를 반드시 제목에 포함 시킬때

검색어 앞에 intitle:을 입력하면 해당 단어가 반드시 포함된 제목을 가진 글을 찾아 보여줍니다.


3. 제외할 검색어를 지정할때

검색어 앞에 마이너스(-)를 입력하면 해당 단어를 제외한 검색 결과를 보여줍니다.


4. 유의어를 검색할때

검색어 앞에 물결표시(~)를 입력하면 해당 단어와 비슷한 의미를 가진 검색 결과를 보여줍니다.


5. 불확실한 단어 검색

정확한 검색어가 떠오르지 않을 때는 문장 사이에 별표(*)를 입력하면 빈 자리를 채워서 결과를 표시합니다.


6. 두 단어 중 아무거나 검색

검색하고 싶은 두 단어 사이에 or를 입력하면 두 단어 중 하나라도 있는 검색결과를 표시합니다.


7. 숫자 범위 지정

검색할 숫자 사이에 마침표 2개(..)를 입력하면 숫자 범위 내의 검색 결과를 보여줍니다.


8. 단어 정의 검색

검색어 앞에 define:을 입력하면 해당 검색어의 정의를 보여줍니다.


9. 특정 홈페이지 안에서만 검색

site:홈페이지주소와 검색어를 함께 입력하면 해당 홈페이지 내에세만 자료가 검색이 됩니다.


10. 특정 홈페이지와 관련 있는 홈페이지 검색

rel:홈페이지주소를 입력하면 해당 홈페이지와 관련 있는 홈페이지를 찾아줍니다.


11. 특정 홈페이지를 링크한 홈페이지 검색

link:홈페이지주소를 입력하면 해당 홈페이지가 링크된 홈페이지를 찾아줍니다.


12. 특정 종류의 파일만 검색

검색어 끝에 filetype:확장자명을 추가하면 해당 확장자의 파일만을 검색합니다.
